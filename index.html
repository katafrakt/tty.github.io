The Ruby toolbox that you can
use for building CLI projects

Features

All too often projects that interact with command line create their own interface logic that gathers input from users and displays information back. Many times utility files are created that contain methods for reading system or terminal properties. Shouldn't we focus our energy on building the actual client? Our time and energy should be spent on creating the tools not the foundation.

  • Table rendering with an intuitive API.
  • Terminal output colorization.
  • Terminal & System detection utilities.
  • Terminal progress bars drawing.
  • Shell user interface.
  • Terminal output paging.
  • Configuration file management.
  • Multilevel logger.
  • Plugin ecosystem.
  • Fully tested with major ruby interpreters.

Quick Start

While installling TTY is simple, there are few requirements. TTY is installed and managed via rubygems, the ruby package manager.

Installation

{% highlight ruby %} $ gem install tty {% endhighlight %}

Data Tables

TTY has component for handling of tabular data and displaying it using flexible rendering capabilities.

{% highlight ruby %} table = TTY::Table.new header: ['h1', 'h2'], rows: [['a1', 'a2'], ['b1', 'b2']] table.render(:ascii) # => +-------+-------+ |header1|header2| +-------+-------+ |a1 |a2 | |b1 |b2 | +-------+-------+ {% endhighlight %}

Shell Interactions

The library provides small DSL to help with parsing and asking precise questions in the shell.

{% highlight ruby %} shell = TTY::Shell.new shell.ask "What is your name?" do argument :required default 'Piotr' validate /\w+\s\w+/ valid ['Piotr', 'Piotrek'] modify :capitalize end.read_string {% endhighlight %}

Progress bars

You can easily indicate progress for various tasks.

{% highlight ruby %} bar = TTY::ProgressBar.new("downloading [:bar]", total: 30) 30.times do sleep(0.1) bar.advance(1) end {% endhighlight %}

Spinners

To easily indicate non-deterministic actions use spinner.

{% highlight ruby %} spinner = TTY::Spinner.new('Loading ... ', format: :spin_2) 30.times do spinner.spin sleep(0.1) end {% endhighlight %}

Screen properties

TTY allows you easily determine screen properties:

{% highlight ruby %} screen = TTY::Screen.new screen.size # => [51, 280] screen.width # => 280 screen.height # => 51 {% endhighlight %}
