Quick Start

While installing TTY is simple, there are a few requirements

TTY components are installed and managed via rubygems, the ruby package manager.

Choose your favourite tty component and install it.

{% highlight bash %} $ gem install tty-prompt Successfully installed tty-prompt Parsing documentation for tty-prompt 1 gem installed {% endhighlight %}

Creating Terminal Apps

You are one step away from generating your terminal application

To create a terminal app you need to install tty gem first.

{% highlight bash %} $ gem install tty {% endhighlight %}

Once tty gem is installed you gain access to teletype executable which will generate your command line application.

{% highlight bash %} $ teletype new app Your teletype project has been created successfully. Run "teletype help" for more commands. {% endhighlight %}
